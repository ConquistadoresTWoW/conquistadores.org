Loot System

Fair and transparent loot distribution for all members

EPGP System (Effort Points / Gear Points)

How It Works:
  • Effort Points (EP): Earned through raid attendance, being on time, and guild contributions
  • Gear Points (GP): Accumulated when receiving loot based on item value
  • Priority Ratio: EP/GP determines loot priority (higher ratio = higher priority)
EP Awards:
  • Raid Attendance: 100 EP
  • On Time Bonus: 25 EP
  • First Kill Bonus: 50 EP
  • Guild Bank Donation: 10-50 EP
  • Helping with M+: 25 EP
  • Officer Duties: 100 EP/week
GP Values:
Item Type Normal Heroic Mythic
Weapons/Trinkets 100 GP 200 GP 400 GP
Armor (Main Pieces) 75 GP 150 GP 300 GP
Accessories 50 GP 100 GP 200 GP
Loot Rules
  • Main spec before off spec
  • Raiders before socials
  • No loot hoarding
  • Tier sets prioritized
  • Weapons distributed fairly
Special Cases
  • New members start with base EP
  • Trials can receive loot after core team
  • GP decay: 10% weekly
  • EP decay: 5% weekly
